To run the file:

This program is written in the python and the below information is how to run this file

Environment Requirements: 
Python version: Python 3.7
Libraries to install: pandas 0.23.4, matplotlib 3.0.1, numpy 1.15.3, statsmodels 0.9.0
IDE: PyCharm IDE

How to run the file:
In this project folder include the product_distribution_training_set.txt, key_product_IDs.txt in the folder.
Now run the main.py file in the PyCharm IDE.

Output: 
After file has done with the predictions the file prints output as “Done with predictions” and write the predictions in a file name output.txt

Note:
Please give at least 15 minutes to run this file due to the large data given.
